# Brand Rules

## Palette

| Token | Hex | Usage |
| --- | --- | --- |
| `navy` | `#1A1A2E` | Primary dark background |
| `ivory` | `#FAFAF7` | Primary light background |
| `red` | `#E63946` | Impact slide and accent |
| `claude` | `#D97757` | Claude and Claude Code mentions |
| `databricks` | `#FF8C00` | Databricks mentions |
| `code-blue` | `#4ECAFF` | Code tags and terminal accents |

## Background Rhythm

```text
Slide 1 -> Navy
Slide 2 -> Ivory
Slide 3 -> Navy
Slide 4 -> Ivory
Slide 5 -> Navy
Slide 6 -> Red
Slide 7 -> Ivory
```

Use one red slide maximum. Reserve it for the most impactful message.

## Typography

- Font family: Syne
- Headings: 800
- Body: 400
- No negative letter spacing

## Non-Negotiables

- Slide size is always `1080 x 1080px`.
- Slide padding is always `88px`.
- Watermark position is always `top: 86px`, `left: -10px`.
- Watermark size is always `108px`.
- Bottom bar is always present and reads `AI · Data Engineering · 2026`.
- Claude and Claude Code mentions use `#D97757`.
- Databricks mentions use `#FF8C00`.
